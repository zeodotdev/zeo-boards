# Zeo Boards

This is a repo of boards made with zeo, made by the team and submitted by users. Your addition of a board to this repo is always welcome.

- demo_board: The board made in the [walkthrough](https://youtu.be/U7OuzNha7m4?si=L84EUU4cbVrLih4U)
- fpv_drone: PCB drone frame designed for DL2205-2300KV Motors. Schematic Only
- jared_tv_remote: Jared's universal TV remote, codename: "Bromote 9000"
- zeo1_debvoard: 1inx1in sensor board built on the esp32c3: componennts placed with ai + autorouted
- zeo1.2_devboard: a more power 1inx1in sensor build built on the esp32s3: companents placed + routed by hand
